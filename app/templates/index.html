Proposta do Site para Atividade Acadêmica

Este site foi desenvolvido como parte da atividade acadêmica do curso de Programador de Sistemas no SENAC-RN. O objetivo desta atividade é colocar em prática os conhecimentos adquiridos durante o curso e demonstrar as habilidades em desenvolvimento web usando Python, Flask, HTML e CSS.

O site possui três páginas principais: "Portfólio", "Sobre Mim" e "Contato". Na página de "Portfólio", é possível ver alguns projetos desenvolvidos. Na página "Sobre Mim", há uma breve descrição do desenvolvedor. E na página "Contato", os visitantes podem enviar mensagens através do formulário de contato.

Além disso, o site é responsivo, adaptando-se a diferentes tamanhos de tela, graças ao uso de Flexbox e estilos CSS adequados.
